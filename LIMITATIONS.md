Limitations of this project:

It only accepts CSV files
* Cannot handle: Excel, JSON, XML, TXT


Data Requirements
* the column names should be specific 
* it is case sensitive in column detection
* requirements are numerical data only (no text values)

* Analysis Features
- basic statistical analysis only, mean, range, std dev
- simple moving average convolution 
- cannot detect specific heart conditions or arrythmias



* Only small datasets

* Time Handling
- it should be sequential
